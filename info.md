# Easy Floorplan

Build an interactive floorplan in Home Assistant with a visual drag-and-drop editor.

- Draw walls (corners snap to close rooms), drop doors and windows that snap to walls
- Add gray furniture diagrams (table, sofa, bed, fridge, stairs, and more) and text labels
- Place entities as icons, live state, or animated **presence ripples**
- Move with the mouse or arrow keys, undo/redo, zoom, set a background color
- Auto-scales to any card or screen size

After install, edit a dashboard → **Add card** → search **Easy Floorplan**.
See the [README](https://github.com/nicosandller/easy-floorplan) for the full
configuration reference.
